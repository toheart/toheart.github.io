使用

使用很简单，分以下几步：

  • 进入项目目录
  • 编写.env文件
  • 退出目录，重启进入，然后就看见一大段输出，选y

以上，完毕；

建议，将PYTHONPATH=/path/to/project也加入到.env中，

并且将项目所需要的环境变量也加入到.env中；

这样我们每次进入项目时，就可以直接运行命令了。
